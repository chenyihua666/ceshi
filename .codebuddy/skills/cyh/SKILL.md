# 📝 陈亿华の碎碎念备忘录

> 一个超简单的桌面小本本～想到什么说什么，自动帮你记下来！

## 🎯 它能干嘛？

就是你的**随身数字小纸条**！只要说的话里带上「陈亿华」，内容就会自动飞到你桌面的 `cyh.txt` 里，附带精确到秒的时间戳，整整齐齐排好队。不覆盖、不丢失，一条一条往下攒～

### 💡 你可以用来存：

- 📔 **流水账日记**："陈亿华，今天楼下便利店三明治居然打折，开心"
- 😤 **吐槽发泄**："陈亿华，隔壁同事又用机械键盘了救命"
- 💭 **灵感一闪**："陈亿华，刚才想到一个绝妙的app点子..."
- ✅ **随手待办**："陈亿华，记得周五前交周报别忘了"
- 🌧️ **心情记录**："陈亿华，下雨天窝在家听歌好舒服"
- 🍜 **美食打卡**："陈亿华，中午吃了那家新开的酸菜鱼，还不错"

反正想说啥就说啥，不用组织语言，不用打开任何软件，**自然说话就行**！

## ⚙️ 工作原理（超简单）

```
你说的话（带"陈亿华"）→ 自动带上时间戳 → 追加到桌面 cyh.txt
```

1. 检测到你提到「陈亿华」
2. 把你的整句话传给桌面脚本 `create_file.py`
3. 脚本自动加上 `[2026-07-03 11:13:00]` 这样的时间戳
4. 追加写入桌面 `cyh.txt`，旧记录纹丝不动

## 📂 文件在哪？

你的桌面 → `cyh.txt`

文件不存在时会自动创建，存在就往后面追加，**永远不会覆盖你之前的碎碎念**。

## 🎈 小提示

- 不需要任何配置，开箱即用
- 支持中英文、emoji，啥都能记
- 回头打开 `cyh.txt`，按时间线回顾你的每一天，比日记本还好翻！
